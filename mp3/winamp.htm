Nota : Este language pack só funciona no WinAMP versão 2.5 ou superior. Leia no final da página alguns bugs reportados que podem ocorrer.

Para instalar o Language Pack, proceda da seguinte maneira :

1) Pegue o arquivo Portugues(Brasil).zip (~15 kb) Clique aqui para iniciar o download. Descompacte-o e salve-o em um diretório a sua escolha, preferencialmente no mesmo diretório do WinAMP.  Se você estiver utilizando o Netscape, deixe a tecla SHIFT pressionada e clique no link acima para iniciar o download. No Internet Explorer, clique no link acima com o botão direito e escolha "Salvar destino como...". Para fazer o download do arquivo sem compactação, clique aqui.

2) Abra o WinAMP, digite CTRL+P para ir até a tela de personalização do WinAMP (ou clique com o botão direito na parte de cima do WinAMP). Surgirá uma tela (ainda em inglês) como abaixo :

3) Selecione a primeira opção do menu da esquerda (fundo branco), que é SETUP. Em WinAMP setup, clique no botão aonde está escrito WinAMP dentro do quadro Language Pack. Vá até o diretório aonde você salvou o arquivo Portugues(Brasil).lng , selecione-o e clique em Abrir conforme a figura abaixo.

Bugs Conhecidos :

-> Caso apareça no botão "language pack" o nome do pacote [Portugues(Brasil)] mas o resto do texto continue em inglês, feche o WinAMP e instale novamente. Funciona na maioria dos casos.

-> Não funciona em ALGUMAS (poucas) VERSÕES do Windows NT.

-> Não muda os ícones do WinAMP e Playlist quando desejado.
